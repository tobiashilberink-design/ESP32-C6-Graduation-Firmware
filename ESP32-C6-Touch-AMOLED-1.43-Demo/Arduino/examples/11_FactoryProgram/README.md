- 编译环境环境是:Arduino IDE V2.3.4  ESP32 Boards V3.3.3 (如果你的环境编译失败,可尝试回退至该版本重新编译)

## 11_FactoryProgram Example Description

- Compilation environment: Arduino IDE V2.3.4, ESP32 Boards V3.3.3 (If compilation fails in your current environment, you can try rolling back to these versions and recompiling.)
- You must refer to the tool configuration in this project directory. See: 11_Fac_Tools_Configuration.png
- This example utilizes LVGL 8. If compilation reports LVGL-related errors, you may try re-installing or replacing the LVGL library.
